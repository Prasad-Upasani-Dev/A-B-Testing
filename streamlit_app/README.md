# Streamlit App - A/B Testing Analysis

Interactive web application for exploring the A/B testing analysis.

## Run Locally

```bash
cd streamlit_app
streamlit run Home.py
```

## Pages

1. **Home** - Project overview and quick metrics
2. **Hypothesis** - Test design and hypotheses
3. **Data Overview** - Dataset exploration
4. **Analysis** - Interactive visualizations
5. **Statistical Tests** - Hypothesis testing results
6. **Decision** - Final recommendations

## Features

- Interactive Plotly charts
- Clean navigation
- Responsive design
- No emojis in content
- Professional styling
